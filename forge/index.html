// core.features
Replaces your entire
dev toolchain
One CLI to build, test, lint, bundle, deploy, and monitor. No more context switching between 7 different tools.
01 / 06
forge.build()
Incremental builds powered by esbuild. Automatic code splitting, tree-shaking, and zero-config TypeScript. Build times under 200ms on most projects.
turbo-mode
02 / 06
forge.test()
Built-in test runner with parallel execution, watch mode, and coverage reports. Compatible with Jest, Vitest, and Playwright out of the box.
parallel-workers
03 / 06
forge.deploy()
One-command deployments to any cloud provider. Automatic preview environments on every PR, with instant rollback capabilities.
multi-cloud
04 / 06
forge.monitor()
Real-time performance metrics, error tracking, and distributed tracing without leaving your terminal. Alerts you before users notice problems.
real-time
